# Restaurant Booking Bot

Telegram-бот для ресторанов.

## Возможности

- Бронирование столиков
- Выбор даты и времени посещения
- Предзаказ блюд
- Просмотр меню
- Управление заказами через админ-панель
- Уведомления клиентов

## Технологии

- Python
- Aiogram
- SQLite
- Telegram Bot API

## Запуск

```bash
pip install -r requirements.txt
python bot.py
